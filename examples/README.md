# Ignition Pay Examples

This directory contains practical examples and reference implementations for building on the Stellar network using Ignition Pay.

## Available Examples

| Example | Language | Description |
|---------|----------|-------------|
| [dart-wallet](./dart-wallet) | Dart | Basic wallet operations |
| [flutter-demo](./flutter-demo) | Dart/Flutter | Cross-platform mobile wallet UI |
| [go-exchange](./go-exchange) | Go | Exchange server with payment processing |
| [go-payment-listener](./go-payment-listener) | Go | Real-time payment monitoring |
| [python-compliance-logger](./python-compliance-logger) | Python | SEP-8 compliance logging service |
| [react-demo](./react-demo) | TypeScript/React | Web-based wallet dashboard |
| [ts-backend](./ts-backend) | TypeScript | Backend services |
| [conformance-test-vectors](./conformance-test-vectors) | JSON | Test vectors |

## Getting Started

Each example has its own README with setup instructions.
